# About This Site

Welcome to my documentation site! This is a centralized location for all my home lab documentation, technical notes, and configuration guides.

## Purpose

This site serves as a comprehensive knowledge base for:

- **Home Lab Configurations**: Documenting my home lab setup, hardware, and network architecture
- **Installation Guides**: Step-by-step instructions for installing and configuring various services
- **Network Documentation**: Network diagrams, IP schemes, and routing configurations
- **Container & Services**: Docker, Kubernetes, and other containerized service documentation
- **Troubleshooting**: Solutions to common problems and issues I've encountered
- **System Restoration**: Procedures for backing up and restoring systems

## Technology Stack

This site is built using:

- **[Jekyll](https://jekyllrb.com/)**: Static site generator
- **[Chirpy Theme](https://chirpy.cotes.page/)**: Clean, minimal theme with excellent features
- **[GitHub Pages](https://pages.github.com/)**: Free hosting and automatic deployment
- **[Markdown](https://www.markdownguide.org/)**: Simple content authoring

## Site Structure

Content is organized using:

- **Categories**: Major topics (Documentation, Tutorial, Docker, etc.)
- **Tags**: Specific topics for easy filtering
- **Archives**: Chronological listing of all posts
- **Search**: Quick access to any content

## Getting Started

- Browse posts by **category** or **tag** using the sidebar
- Use the **search function** to find specific topics
- Check the **archives** for a chronological view
- Read the [Welcome post](/posts/welcome-to-documentation-site/) for an introduction
